# What This Template Does

This template is a project operating pack.

## It does
- give the project a stable file structure
- preserve the roadmap, checklist, intake standard, and logs
- provide a repeatable handoff/import workflow
- reduce drift by keeping file destinations explicit

## It does not
- act like autonomous automation
- guarantee that every future reply will include files
- replace direct working instructions in the conversation

## Practical effect inside a new project
If this pack is added to the project files, it becomes the reference system for how work should be organized and routed. It helps preserve the workflow across project moves.
