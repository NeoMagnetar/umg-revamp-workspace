## Evidence Log Rules
- Every uploaded batch gets a batch ID
- Every file gets a disposition
- Every file gets roadmap tags
- Every nontrivial insight gets assigned to a revamp phase
- Duplicate insights are merged, not repeated
- Contradictions are explicitly flagged
- Strong candidate text for docs/spec/PRD is marked EXPORT_CANDIDATE
