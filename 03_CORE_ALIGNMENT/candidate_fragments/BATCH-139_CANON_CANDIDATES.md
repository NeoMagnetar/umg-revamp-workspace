# BATCH-139 — Canon Candidates

## Status Note
These are candidate canonical fragments extracted from a derived terminology-governance summary.
Inclusion here does not itself promote them to canon.

## Candidate Fragments
1. A self-map is an explicit declaration that UMG has reviewed, accepted, and frozen a term unchanged.
2. Even identity relationships must be declared.
3. Mappings are reversible lexical bridges, not semantic rewrites.
4. Mapped terms must preserve authority, governance, and resolution semantics.
5. Release mappings should carry explicit invariants and deprecation controls.

## Promotion Caution
Before canon lock, compare these fragments against:
existing mapping canon,
glossary and terminology policy files,
later control-room decisions on release-manifest structure,
and any validator implementation constraints.
