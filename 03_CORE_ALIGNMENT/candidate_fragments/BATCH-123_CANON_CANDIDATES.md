## Candidate Fragments
1. UMG is a Cognitive Specification Layer.
2. MOLT is the typed grammar of UMG.
3. Canonical MOLT order: Trigger, Directive, Instruction, Subject, Primary, Philosophy, Blueprint.
4. MOLT hierarchy is semantic ordering, not automatic suppression.
5. Governance is the only mechanism that overrides.
6. No governance = no suppression.
7. Type ≠ Role.
8. Bundle preserves structure without changing semantics.
9. Merge is explicit synthesis with preserved source traceability.
10. Off is a state or role, not a MOLT type.
11. Snap is UX-only and has no compiler semantics.
12. Scope is explicit and deterministic.
13. NeoBlock is the compiled representation of one resolved Molt Stack.
14. NeoStack is a domain-level grouping, not a hierarchy.
15. Compiler emits RuntimeSpec and Trace.
16. No Trace = not UMG.
17. Determinism requires stable ordering, explicit rules, and no hidden randomness.
