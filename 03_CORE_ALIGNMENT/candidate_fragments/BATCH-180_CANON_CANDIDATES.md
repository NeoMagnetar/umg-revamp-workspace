## Candidate Fragments
1. One grouped JSON file should contain a list of full block objects, not strings or category summaries.
2. Each intended row/item must become its own block when the agreed pattern is row-level blockification.
3. No alternate packaging mode unless explicitly requested.
4. Artifact generation should be linted against importer expectations before release.
5. Surface the exact filename and exact unpack command together.

## Promotion Caution

current importer implementation,
later compiler/QA decisions,
formal template canon,
and any stabilized naming conventions adopted elsewhere.
