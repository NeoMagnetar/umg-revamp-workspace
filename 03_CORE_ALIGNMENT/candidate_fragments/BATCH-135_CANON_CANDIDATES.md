# BATCH-135 — Canon Candidates

## Status Note
These are candidate canonical fragments extracted from a derived block-architecture summary.
Inclusion here does not itself promote them to canon.

## Candidate Fragments
1. A serious block deserves its own internal architecture.
2. Adapters translate shape; blocks enforce meaning.
3. Observation must precede interpretation.
4. Process is protection.
5. Presentation is a contract.
6. Semantic render structure and renderer/UI typography are different layers.
7. Block sections must be explicit to the block’s own function.
8. An agnostic sleeve can host specialized skill cartridges without changing the whole base system.

earlier compiler and Studio batches,
current glossary language,
future render-spec decisions,
and any control-room decisions about artifact-tree legality.
