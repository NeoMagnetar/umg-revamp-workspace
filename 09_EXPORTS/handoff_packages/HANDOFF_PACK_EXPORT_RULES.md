# Handoff Pack Export Rules

## Purpose
Define how file sets should be packaged for transfer into the workspace.

## Rules
- Prefer one zip package per response when multiple related files are produced.
- Keep paths inside the package aligned to workspace destinations when possible.
- Include a short import note or script when placement is non-obvious.
- Log exported artifacts in the relevant checklist, synthesis log, or status log.
- Avoid packaging raw intake and finalized canonical files together unless the bundle is explicitly an intake memo pack.
