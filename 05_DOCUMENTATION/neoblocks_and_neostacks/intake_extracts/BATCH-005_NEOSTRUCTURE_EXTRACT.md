- The batch names both cognition-facing and blueprint-facing NeoBlocks, including governance and format-routing blocks.
- Output-shaping blueprints are better modeled as blueprint blocks or a router block than as full cognition stacks.

  - Art of War

  - UMG Studio workmode

## Architecture Value
This batch is important because it clarifies a practical architecture split:
- sleeve = top-level configuration
- stacks = cognition/work layers
- NeoBlocks = compiled active units
- blueprints/router = response emission layer
