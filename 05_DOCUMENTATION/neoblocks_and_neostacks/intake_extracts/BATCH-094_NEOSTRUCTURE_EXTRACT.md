## Candidate NeoStack Layer
### Diplomatic Layer
Source describes a new stack layer with the following properties:
- low-profile
- non-default active
- conditionally invoked

## Structural Importance
This batch pressures UMG to define whether:
- layers are first-class architectural units
- protocol blocks can live inside situational layers
- layers can act as conditional overlays
- conditional layers change execution scope without redefining baseline stack identity

## Open Structural Tensions
- relation of layer to stack, bundle, and block remains underspecified
- whether a protocol layer is unique or one of many possible conditional layers
- how conditional layer composition interacts with other stack semantics
