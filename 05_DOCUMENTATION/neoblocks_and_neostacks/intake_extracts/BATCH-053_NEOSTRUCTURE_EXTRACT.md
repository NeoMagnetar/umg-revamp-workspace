## Structural Patterns Surfaced

### Dual-template model
- classic MOLT for clarity, stability, and fast prototyping
- Skeleton Agent for adaptation, runtime evolution, and more AGI-like simulation

### Alignment architecture layers
- Mainline as universal anchor
- Alignment Block as inheritable local lineage layer
- Tether as limited context-sharing link among a small number of agents

### Multi-agent relay architecture
- specialist handoffs
- backend/frontend mind split
- reasoning separated from presentation
- sequence-based agent collaboration instead of one monolithic bot

### Product framing
- Citadel-style multi-room / multi-agent environment
- research-first staging before broad app build
- eventual user-buildable UMG agents

## Structural Tensions

- whether Tether is a protocol object, block, or system-layer construct
- whether universal alignment should live primarily in Mainline or in inheritable agent-local structures
