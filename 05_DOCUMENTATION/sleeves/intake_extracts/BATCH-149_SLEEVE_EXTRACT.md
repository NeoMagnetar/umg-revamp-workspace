# BATCH-149 — Sleeve Extract

## Scope
Sleeve-equivalent identity layering and resleeving signals extracted from the UMG Mind Stack source.

## Sleeve-Equivalent Signal
The source does not formalize a full sleeve system, but the Mind Stack clearly behaves like a sleeve-equivalent identity package composed from:
- VaultHeader
- AlignmentCore
- PhilosophyLayer
- MOLTCore
- MemoryConfig
- LedgerBlock

## Resleeving Signal
Different bots such as Mercury and Sun Tzu imply different identity packages.
Resleeving is suggested through swapping identity-bearing layers such as:
- VaultHeader
- Philosophy
- Tone
- other role-defining blocks

## Structural Doctrine Signal
- chatbot identity should be layered, not monolithic
- identity can be recomposed by changing selected layers
- runtime cognition depends on stacked configuration rather than one prompt
