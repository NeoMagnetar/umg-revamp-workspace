# BATCH-076_BLOCKS_MOLT_EXTRACT

## Scope
MOLT roles, NeoBlocks, and bundled execution units.

## Extracted Signals
- MOLT is the atomic unit of meaning/instruction/logic/style.
- Roles explicitly named:
  - Primary
  - Subject
  - Philosophy
  - Instruction
- NeoBlocks are named under NB.* conventions.
- Example NeoBlocks:
  - NB.GOV.POE_UMG.CORE.v1
  - NB.TRUSTED.ALLY.PROVISIONAL.v1
  - NB.RECURSIVE_TEST.PASS1
- NeoStacks operate as bundled loaded context.
- Bundle-like behavior appears through Active Stack and grouped sleeve/governance execution.

## Interpretation Notes
- This batch reinforces core role taxonomy without changing the basic four MOLT roles.
- The strongest extension here is not a new role but stronger packaging around governance, NeoBlocks, and runtime bundle state.
