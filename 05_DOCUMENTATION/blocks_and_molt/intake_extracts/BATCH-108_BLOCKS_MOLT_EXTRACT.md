## Strongest stable signals
### Primary
- Mandatory root block.
- Nothing goes above it.
- Central anchor for stack growth.

### Subject
- Domain tags and scope notes.

### Instruction
- Rules and constraints.

### Philosophy
- Tone and ethical lens.

### Blueprint
- Layout / format / rendering hints.
- Separate from executable payload.

### Directive
- Optional strategic overlay.

### Trigger
- Execution condition.
- May be cron-like or manually set in concept.

## Structural rules tied to MOLT behavior
- Vertical = stack / hierarchy / priority.
- Horizontal = overlay / merge candidate behavior.
- Role semantics should drive graph behavior.
- Category organization helps browsing, but stack behavior remains semantic.
