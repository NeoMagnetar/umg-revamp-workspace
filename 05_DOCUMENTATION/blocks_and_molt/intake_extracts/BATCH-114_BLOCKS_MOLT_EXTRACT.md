# BATCH-114 — Blocks / MOLT Extract

## Stable MOLT Signals
- Blocks are semantic units that mean something but do nothing on their own.
- Each block answers exactly one cognitive question.
- The batch presents the following fixed role set:
  - Primary
  - Subject
  - Directive
  - Instruction
  - Philosophy
  - Blueprint
  - Trigger

## Canon-Tight Role Mapping
- Primary → Why
- Subject → About what
- Directive → How (strategy)
- Instruction → Under what rules
- Philosophy → Under what values
- Blueprint → In what form
- Trigger → When eligible

## Structural Rules
- Types are described as:
  - fixed
  - non-mergeable
  - non-reorderable
- Vertical = authority / constraint
- Horizontal = coexistence

## Composition Rules
### Merge
- Only mechanism by which meaning fuses
- Explicit
- Traceable
- Produces a new block
- Does not mutate source blocks

### Bundle
- Horizontal grouping
- No new meaning
- No new authority

## Important Boundary Rules
- Trigger does not add meaning or authority and cannot bypass governance.
- Priority is local to conflict sites and ignored when no conflict exists.
- Governance is supreme and OFF is the only exclusion mechanism.
