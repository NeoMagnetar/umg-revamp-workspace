## Role Triad Signal
This batch provides one of the clearest lightweight explanatory triads in the intake set:
- Primary = core intent / what you want
- Merge = context/data / what to include
- Blueprint = style/strategy / how it should feel

This matters because it is compact, memorable, and useful for onboarding or public explanation.

## Modular Block Signal
The source treats these roles as:
- composable
- independently editable
- swappable
- plug-and-play

That strengthens the argument that UMG blocks are not only conceptual labels, but modular units that can be recomposed without rewriting the entire reasoning structure.

## Structural Tensions

- Merge is strong here, but Priority, Trigger, Philosophy, and other roles are underdefined or absent
- Blueprint is clearly a rendering/strategy layer here, which may only partially overlap with broader blueprint usage elsewhere
- no explicit block schema fields are provided in this batch
