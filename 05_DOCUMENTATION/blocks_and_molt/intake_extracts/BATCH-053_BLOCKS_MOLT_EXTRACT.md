## Functional Distinctions

### Instruction
Current action rules; strongest compact utility.

### Merge
Conflict-resolution and combination logic; not equivalent to Tether or Skeleton.

### Meta
Self-regulation / self-checking layer rather than ordinary action execution.

### Format
Rendering or presentation layer, separated from reasoning.

### Memory
Retention policy or persistence logic.

### Off
Scoped disabling logic that may be replaceable by more precise instructions.
