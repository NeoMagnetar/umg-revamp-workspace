# BATCH-018 — Blocks / MOLT Extract

## Atomic Layer
Preferred term:
- MOLT block

Reason:
- clarifies the atomic semantic layer
- avoids ambiguity around generic “block”

## Role Families Used
- Trigger
- Directive
- Instruction
- Subject
- Primary
- Philosophy
- Blueprint

## Strongest Working Refinements
### Trigger
Should be treated more like a gate than a generic content block.

### Directive
Acts as a route selector that can open different paths.

### Primary
One-primary discipline remains important even when many instructions or subjects are active.

### Blueprint
Output shaping only; not the source of runtime truth.

## Structural Rule Candidates
- MOLT blocks are the atomic semantic units.
- Active gates open routes.
- Routes light relevant MOLT blocks.
- Relevant MOLT blocks compress into NeoBlocks.
