# BATCH-182 - Blocks and MOLT Extract

## Scope
Block- and MOLT-structure signals extracted from the Builder runtime source.

## Active Builder-Facing MOLT Taxonomy
- Primary
- Subject
- Instruction
- Philosophy
- Blueprint
- Directive
- Trigger
- Deployment
- Off

## Operational Role Meanings
- Primary = anchor or main plan intent
- Subject = focus domain or topic
- Instruction = actionable execution logic
- Philosophy = worldview or alignment overlay
- Blueprint = layout or render logic
- Directive = strategic modifier
- Trigger = event-based activation logic
- Deployment = export or runtime target logic
- Off = retained but disabled logic

## Structural Signal
Array order and stack priority matter within these MOLT groups.
Builder logic assumes that MOLT role, stack ordering, cantocore, snap metadata, and merge metadata together produce downstream runtime behavior.

## Important Caution
Meta also appeared as a Builder extension concept, but it is not confirmed as long-term canonical MOLT taxonomy from this batch alone.
