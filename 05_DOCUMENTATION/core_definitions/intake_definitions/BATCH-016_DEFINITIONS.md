# BATCH-016_DEFINITIONS

## Terminology Extract

### Sleeve
- Structured control layer that defines how an AI system thinks, prioritizes, and behaves before response generation.
- Governed cognitive configuration.
- Determines who the system is right now.
- More comparable to a loadable operating profile than a persona prompt.

### Governance
- Pre-response control layer.
- Distinct from decorative style or surface framing.
- Can remain logically active even when not visibly declared.

### Visible Governance
- Governance as surfaced through active-state declaration, reporting, or output-visible control markers.

### Latent Governance
- Governance that remains structurally or conceptually active without being clearly surfaced in output/state display.

### Envoy
- Governed agent instance.  
- External-facing translation candidate: **Agent**.

### Cognitive Runtime Container
- External-facing translation candidate for **Sleeve**.
- Approximate term only; not identical to a software container.
