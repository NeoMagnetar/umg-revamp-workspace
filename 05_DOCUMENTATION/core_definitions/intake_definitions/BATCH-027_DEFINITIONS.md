# BATCH-027 — DEFINITIONS EXTRACT

## Terminology
### Sleeve
- Domain operations sleeve.
- Defines working identity and active operational scope.
- Not treated as a generic helper prompt.
- In this batch, framed as an Outlands Razor operations assistant.

### NeoStack
- Coherent domain cognition layer for a specific operational area.
- Owns a defined problem surface such as setup, script install, authoring, troubleshooting, or compliance awareness.
- Activated according to user intent and task type.

### NeoBlock
- Primary compiled functional unit within a stack.
- In this batch, implied as the dominant operational block per stack rather than a giant blended response persona.
- Still preliminary in this source.
