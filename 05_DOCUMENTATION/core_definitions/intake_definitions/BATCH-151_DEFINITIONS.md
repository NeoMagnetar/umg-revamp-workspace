# BATCH-151 — Definitions Extract

## Terminology Scope Note
These definitions are extracted from a derived schema-and-engine summary.
They remain Stage 1 evidence until explicitly promoted elsewhere.

## Core Canon Terms
**MOLT**  
Modular Operating Language of Thought.

**UMG**  
Cognitive Operating System.

**Atomic Unit of Thought**  
Core phrase used to describe a block as the smallest composable cognition unit.

## Engine Terms
**Snap**  
Core engine operation for fitting blocks together vertically or horizontally according to structural compatibility.

**Merge**  
Core engine operation for combining compatible same-type blocks using additive, override, or layered strategies.

**Stack**  
Core engine operation for ordering hierarchical influence, with top-down strength and optional overlay behavior.

## Role-Separation Terms
**Instruction**  
Logic or constraint layer.

**Philosophy**  
Ethics or tone layer.

**Blueprint**  
Formatting or layout layer.

**Directive**  
Strategy or mutation layer.

## Definitional Candidates
MOLT = atomic unit of thought.  
UMG is a Cognitive Operating System.  
Blocks define both content and behavior.  
One block, one MOLT role.
