## Architecture / System Terms

- Described as a **modular cognitive architecture** for building intelligence from editable, composable logic blocks.
- In this batch, UMG functions as the **base logic substrate** for higher layers.
- Status: candidate descriptive language, not finalized canon.

### CEL (Creative Extension Layer)
- Described as a modular creativity layer extending UMG beyond logic composition into creative cognition.

  - mutation / blend
  - creative expansion
- Status: candidate extension-layer concept.

- Described as an internal evaluation / filtering mechanism for CEL outputs.
