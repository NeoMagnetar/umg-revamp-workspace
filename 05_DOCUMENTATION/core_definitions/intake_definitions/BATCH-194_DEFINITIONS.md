## Terminology Scope Note
These definitions are extracted from a derived Bolt-integration summary.

## Core Terms
Canonical Builder App Path
The single authoritative app location from which the builder should be served, edited, and reasoned about.

Block JSON Naming Convention
Required relationship among block IDs, file names, and runtime lookup paths so the builder can load blocks deterministically.

Builder-First Route
Routing stance in which the visual block builder is the default user entry point.

Landing-First Route
Routing stance in which marketing or explanatory content is the default user entry point and the builder is secondary.
