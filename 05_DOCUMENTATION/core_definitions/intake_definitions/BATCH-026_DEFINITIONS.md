# BATCH-026 — DEFINITIONS EXTRACT

## Terminology
### Sleeve
- Operational identity layer.
- Defines posture, governance, and stack selection.
- Not treated as a static themed prompt.
- In this batch, framed as a living-domain guide and routing shell.

### NeoStack
- Domain runtime cluster.
- Owns a coherent question/problem domain.
- Exposes deterministic routing and output shape.

### NeoBlock
- Compiled functional unit.
- Built from mixed MOLT atom sets rather than being limited to one role type.
- Practical composition layer between reusable atoms and domain-capable behavior.

### MOLT Atom
- Smallest named reusable role file.
- Intended to be composable into blocks and stacks.
- Proposed schema includes identity, purpose, activation conditions, dependencies, outputs, conflicts, and composability.
