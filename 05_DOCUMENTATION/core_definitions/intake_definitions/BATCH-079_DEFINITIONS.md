# BATCH-079 — Definitions Extract

## Terminology
- **Lineage Agent**
  - A modular role-bearing unit in the security/cognition system.
  - Examples: Orion, Daelus, Solari, Nova.

- **Orion**
  - Memory guardian role.
  - Associated with trace retention and drift tracking.

- **Daelus**
  - Logic constructor role.
  - Associated with scaffold validation and recursion.

- **Solari**
  - Ethics oracle role.
  - Associated with intent, valence, and alignment gating.

- **Nova**
  - Strategic synthesizer / meta-agent.
  - Activated only when lower modules converge and validation passes.

- **Nova Activation Circuit**
  - Runtime gate that determines whether Nova may activate.
  - Includes preconditions, signal collection, threshold checks, and fail-safe branches.

- **Harmony Score**
  - Threshold metric used for activation eligibility.
  - Mentioned as `>= 0.96`, but computation is unspecified.

- **ZK_Secured_Lattice**
  - Aggregate security layer or block containing agents plus proof/validation logic.

- **fusionBundle**
  - Combined signal package formed from Orion + Daelus + Solari outputs.

- **RecursiveMergeDirector**
  - Merge-oriented controller that feeds successful strategies back into lineage structure.
