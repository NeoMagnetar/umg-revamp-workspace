# BATCH-050 — Definitions Extract

## Role Language
- **Trigger**: activation/routing condition; external gate; should not author identity or rewrite core invariants.
- **Directive**: intent-shaping control atom within grouped intent.
- **Instruction**: action / execution guidance within grouped intent.
- **Subject**: target/topic/object of reasoning within MOLT layer.
- **Primary**: dominance / salience carrier used in deterministic conflict ordering.
- **Philosophy**: governing reasoning stance or constraint posture.
- **Blueprint**: output/render planning layer distinct from cognition.

## Architecture Language
- **Sleeve**: full current identity / operating frame; contains governance-linked posture, active stacks, defaults, constraints, triggers, weights, merge order, and expression limits.
- **NeoStack**: domain container.
- **NeoBlock**: grouped intent unit containing one or more MOLT atoms.
- **MOLT block / atom**: smallest semantic primitive.

## Runtime / Compiler Language
- **Deterministic dominance ordering**: output/selection ordering based on effective weight, axis rank, merge rank, and stable identifier.
- **Style overlay**: surface-only modulation; should not write Directive / Instruction / Primary / identity fields.
- **Mutation policy**: formal rules defining what can change, what cannot change, and what causes rejection.
- **Validation schema**: machine-checkable constraints for sleeve / stack / block / atom integrity.
- **Behavioral Abstraction Model**: publication-safe label for conceptual model behavior mapping without claiming backend introspection.
- **Interchange / wire format**: likely linear MOLT representation used for portability and compiler compatibility even if richer overlays exist.

## Governance Language
- **Governance precedence**: outer/base/platform safety above sleeves; sleeves above stacks; stacks above blocks; expression below intent.
- **Tool Anchor Ethos**: AI as computational reasoning tool; humans as sovereign choosers; bounded procedural agency; no moral sovereignty claim.
- **Bounded procedural agency**: delegated, constrained, auditable agency rather than autonomous or moral authority.

## Surface / Output Language
- **Format ≠ cognition**
- **Presentation ≠ execution**
- **Meta diagnostic summary ≠ internal reasoning chain**
- **Mode selects cognition; blueprint/output mode selects emission** (concept retained from adjacent project canon)

## Definitional Candidates
- NeoStacks = domain containers
- NeoBlocks = grouped intent
- MOLT blocks = atomic primitives
- Mythos can assist compression/anchoring but must remain subordinate to governance
