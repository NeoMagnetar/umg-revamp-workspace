## Core Terms
Cognitive Specification Layer
A layer that specifies what cognition should exist rather than how cognition runs. In this source, UMG is explicitly locked as a tech-agnostic CSL.

RuntimeSpec
The structured compiled artifact describing what should be active, constrained, allowed, or shaped after UMG compilation.

Trace
The deterministic and mandatory audit record of how the final compiled structure was derived.

PrimaryShell
The single output anchor per snap, defined as a role rather than a MOLT type.

## Structural Terms
Domain
The leading human-facing candidate term for grouped behavior, concern, or capability collections, explicitly separated from mechanical compiler stack behavior.

Stack
Mechanical same-type priority resolution system using stack_key and stack_rank.

Merge
Explicit same-type operation that combines compatible blocks with the same non-empty merge_key.
