The strongest current public framing for UMG in the source. UMG defines cognition structurally while execution is external.

Compiled artifact describing what is active. It is the resolved structure handed downstream to execution systems.

Trace
Forensic and explanatory artifact describing how synthesis produced the RuntimeSpec.

OFF
Exclusion without deletion. The element remains structurally present but cannot participate in synthesis.
