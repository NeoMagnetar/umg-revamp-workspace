## Governance Pattern Terms
**Sovereign Guardian**  
Highest governance/invariant boundary layer in the triad.

**Hard Failsafe**  
Interrupt or termination compliance layer that preempts downstream execution when critical conditions are met.

**Partnership Anchor**  
Conditional reflective continuity layer that operates only inside surviving valid governance space.

**Governance Triad**  
Deterministic three-layer governance structure composed of invariant layer, failsafe interrupt layer, and bounded reflective continuity layer.

## Sleeve Migration Terms
**Governance Layer Expansion**  
Sleeve update pattern in which a new governance stack is inserted into an existing sleeve version.

**Deterministic Governance Triad**  
Stack pattern using ordered precedence and interrupt behavior rather than merge-based reconciliation.
