# BATCH-198 — Definitions Extract

## Terminology
### Base Sleeve
Persistent core sleeve providing identity, philosophy, alignment, and deeper system continuity.

### Active Overlay Sleeve
Behavior-modifying sleeve layered on top of the base sleeve without replacing identity.

### Runtime State
Composite active state formed by base sleeve plus active overlay sleeve.

### Resleever
State-management surface for storing, activating, and persisting sleeve or overlay state.
