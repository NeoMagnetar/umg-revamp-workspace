## Terminology Scope Note
These definitions are extracted from a derived comparative-architecture summary.

## Core Terms
Cognitive Specification Layer
A layer that structures cognition and governance explicitly rather than merely prompting or executing behavior.

Agent World Model
Architecture that centers structured internal world representation, state dynamics, transition modeling, and predictive planning.

Governed Predictive Agent Architecture
Provisional phrase for a hybrid architecture where UMG supplies governance and cognitive grammar while a world-model component supplies state dynamics and future prediction.

Structured Cognition Over Time
Phrase used to describe UMG as more than a static prompt structure, while still not yet formalized as a full probabilistic transition model.

## Design-Space Terms
State NeoBlock
Proposed future block class for explicit state representation.

Transition NeoBlock
Proposed future block class for explicit transition modeling between states.

Simulation NeoBlock
Proposed future block class for future trajectory or scenario modeling.
