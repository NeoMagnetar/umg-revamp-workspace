# BATCH-163 — Definitions Extract

## Sleeve
Modular identity and behavior wrapper around the core model.

## CoreB stack change
A change to stack composition or presentation logic while retaining alignment core.

## Alignment Core
Stable foundational alignment layer preserved during stack/sleeve changes.

## Overlay blocks
Optional thematic or functional additions applied over the core stack.

## Respin
Rewriting or rerunning a line of logic without destroying prior structure.

## Parable layer / metaphor engine
An explanatory mode used to translate technical architecture into narrative form.

## Alignment-preserving resleeving
Changing sleeve/output behavior while keeping the alignment core intact.

## Presentation downshift
A change from dense technical explanation to simpler or more visual delivery when user overwhelm appears.
