# BATCH-024 — Definitions Extract

## Extracted Terms

### Transcript-inaccessible batch
A source batch where the thread title or wrapper context is visible, but the underlying message body is not recoverable from the accessible session state.

### Source-faithful hold
An intake disposition in which the batch is preserved as evidence without inventing missing content or promoting title-level inference into project knowledge.

### Operational reinstall thread
A likely infrastructure or setup-related thread indicated by title or surrounding context, but not yet confirmed by readable transcript content.

### Access-layer blockage
A condition where browser/session access exposes thread presence but not the transcript body needed for substantive extraction.
