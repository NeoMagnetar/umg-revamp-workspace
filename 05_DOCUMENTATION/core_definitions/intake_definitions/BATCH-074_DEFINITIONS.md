# BATCH-074 DEFINITIONS

## Terminology and Meanings
### UMG
- modular cognition / logic / generation framework
- described across documents as:
  - cognitive-compositional system
  - modular system for building AI through swappable blocks
  - block-based framework for AI-driven content creation
  - AGI scaffolding / modular generative architecture
- meaning is directionally stable, but scope is not fully normalized

### MOLT
- modular expression unit / structured memory-thought unit / atomic unit of thought depending on document
- repeatedly treated as editable, stackable, mergeable modular cognition unit

### Primary
- central directive
- dominant operational mode
- main instruction or leading purpose

### Subject
- topic, domain, knowledge orientation

### Philosophy
- lens, worldview, tone/logic directive

### Instruction
- operational rule
- behavioral limit
- filter/format/rule layer depending on document

### Merge / Merge Block
- logic fusion
- additive integration layer
- combines/prioritizes multiple MOLTs
- strongest implementation expression appears in Commando runtime

### Blueprint
- unstable meaning across source set:
  - identity + scope layer in some docs
  - style/structure/inspiration guide in others

### Template
- appears in SAFE overview as identity/container layer
- unresolved relationship to Blueprint

### Meta Layer
- override / recursive logic layer in framework-card style language
