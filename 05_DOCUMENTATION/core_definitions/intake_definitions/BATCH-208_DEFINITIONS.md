# BATCH-208 — Definitions Extract

## Terminology
### VaultHeader
Identity/origin/version wrapper for a UMG-rendered module.

### Alignment.Core
Ethics, rules, or constraints layer applied to a rendered plan/module.

### Instruction.Layer
Operational flow, governance, or protocol layer.

### Overlay / Overlay.Core / Overlay.Stack
Contextual lens layer such as zoning, airflow, hazard, privacy, or similar contextual framing.

### Threshold MOLT
Transition zone between systems or environments.

### Green Node / Memory Core
High-value park, anchor, or civic node treated as a system center.

### Connector MOLT / Flow Spine
Circulation or linking structure.

### Resource Zone / Water MOLT
Resource-bearing or service-bearing spatial zone.

## Domain-Specific MOLT Language
### Residential MOLT
Housing or domestic zone in a civic/site reinterpretation.

### Access MOLT
Entry or access structure such as a driveway node.

### Vertical Logic / Stair Node
Vertical circulation module in floor-plan interpretation.

### Hearth Anchor
Emotionally central domestic anchor.

### Rest Node / Private MOLT
Privacy-oriented room or zone.

## Output Layer Language
### Raw Technical Blueprint
Original civil/architectural source document intended for contractor or engineering precision.

### UMG Cognitive Blueprint
Modular cognitive rendering optimized for conceptual clarity, modular logic, and agent training.

### Hybrid Field Layer
Combined UMG + standards/engineering output intended to bridge cognition and field precision.

## Definitional Candidates
- source plan → modular reinterpretation → structured layers → labeled blocks → optional redraw
- UMG layer = conceptual mapping and modular logic; standards layer = field precision
- preserve raw source and generate a distinct cognitive blueprint layer
