# BATCH-090 — Definitions Extract

## Terminology
### UMG
A modular, composable cognition framework that structures thought through editable blocks rather than a single monolithic prompt.

### Primary
The block that interprets or defines user intent; also described as the main directive or core content.

### Merge
The block that incorporates supporting information, context, data, rules, constraints, and required details.

### Blueprint
The block that applies style, tone, philosophy, structure, or strategic lens without being the direct content payload.

### UMG Engine / Parser
Conceptual orchestration layer that interprets blocks and converts them into usable model instructions.

### Agent
Specialized downstream execution unit such as Research Agent, Planning Agent, or Persona Agent.

### Metacognition
Higher-order reflective layer mentioned as depth around the core block model.

### Portable Thought Objects
Public-facing phrase describing modular, reusable thought structures.

### Orchestra of Thought
Communication metaphor: user as conductor, Primary as melody, Merge as harmony, Blueprint as style, agents as instruments.

## Definitional Candidates
- Primary / Merge / Blueprint remains the canonically foundational trinity.
- UMG turns prompt engineering into a construction process.
- UMG enables composable cognition.
- UMG provides explicit reasoning layers.
- UMG creates portable thought objects.
