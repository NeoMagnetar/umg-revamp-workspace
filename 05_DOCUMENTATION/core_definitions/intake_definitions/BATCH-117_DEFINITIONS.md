# BATCH-117_DEFINITIONS

## Terminology Extract
- **KinetiAGI (local comparison context):** treated as an external epistemic governance doctrine described by screenshot text, not a fully verified literature-backed source in this batch.
- **UMG:** modular cognitive control/specification substrate making policy enforceable and auditable through explicit artifacts and governance.
- **epistemic mode:** representational label separating exploratory, evaluative, or endorsed output states; proposed here as a mapping device, not canon.
- **Explore / Assess / Endorse:** suggested mode labels for explicit epistemic posture; presented as implementation mapping, not ratified UMG canon.
- **audit tuple:** mandatory audit package attached to assessment/endorsement outputs, including participation, exclusion, conflict, and evidence fields.
- **failure-first bundle:** grouped checks required before endorsement, including falsification or adversarial review steps.
- **constraint-first evaluation:** governance/order rule requiring tests and constraints to run before confidence/endorsement.
- **policy layer:** external doctrine or epistemic requirement set.
- **UMG substrate / control plane:** modular artifact system and governance structure that enforces declared policy.
