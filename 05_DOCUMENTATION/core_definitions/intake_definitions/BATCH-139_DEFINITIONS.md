# BATCH-139 — Definitions Extract

## Terminology Scope Note
These definitions are extracted from a derived terminology-governance summary.
They remain Stage 1 evidence until explicitly promoted elsewhere.

## Mapping Governance Terms
**Reversible Lexical Mapping**  
A translation bridge between academic-safe and UMG terminology that preserves semantics rather than rewriting them.

**Self-Map / Identity Map**  
Explicit declaration that a reviewed term remains unchanged under the mapping system while still participating in reversibility and governance.

**Collision-Free Mapping**  
Mapping surface in which each governed entry remains injective/reversible according to the release rules in effect.

**Deprecation Ledger**  
Artifact proposed for recording retired labels, replacements, and release-stage mapping history.

## Release and Validation Terms
**Mapping Freeze Manifest**  
Release artifact describing mapping version, status, invariants, reversibility requirements, and deprecation controls.

**Identity Tagging**  
Proposed metadata indicating that a mapping entry is a reviewed identity relationship rather than an omitted row.

## Artifact-Harvest Terms
**Terminology Decision Packet**  
Compiled artifact containing mapping rationale, invariants, and acceptance logic.

**Collision Proof**  
Artifact demonstrating absence of ambiguous reverse mapping under a given release surface.

## Definitional Candidates
A self-map is an explicit declaration that UMG has reviewed, accepted, and frozen a term unchanged.  
Even identity relationships must be declared.  
Mappings are reversible lexical bridges, not semantic rewrites.
