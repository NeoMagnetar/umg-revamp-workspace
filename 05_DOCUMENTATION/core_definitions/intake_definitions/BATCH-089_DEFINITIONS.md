# BATCH-089 — Definitions Extract

## Terminology
### MCA
Modular Cognition Agent; draft candidate classification for UMG-based systems positioned as an alignment-first alternative framing to AGI.

### UMG
Modular, composable cognition framework built from blocks and layers such as memory, alignment, instruction, and output systems.

### Negentropy
Conceptual framing of intelligence as maintained structure, coherence, and order against entropy across time.

## Role Language
### Subject
Domain/type/orientation block; MCA is introduced as a possible Subject-type classification.

### Instruction
Rules or constraints, including hidden internal instruction blocks that alter reasoning lens.

### Philosophy
Tone, worldview, or reasoning stance.

### Alignment
Constraint/governance layer that shapes good-vs-bad framing and bounded behavior.

### Memory
Stabilizing continuity layer supporting self-consistency and history-aware cognition.

### CEL
Voice/output or creative layer participating in modular cognition.

## Architecture Language
### Hidden Instruction Block
Internal lens modifier that affects reasoning without explicit surface declaration.

### Compositional Identity
Identity formed through active stack composition rather than fixed monolithic essence.

### Stack
Layered runtime state including cognition, memory, alignment, instruction, and voice/output components.

## Definitional Candidates
- MCA = alignment-first alternative to AGI framing for UMG-based systems
- Intelligence = maintaining order through modular composition
- Compositional identity
- Hidden instruction block
