# BATCH-195 - Definitions Extract

## Terminology Scope Note
These definitions are extracted from a derived CSL/compiler-foundation summary.
They remain Stage 1 evidence until explicitly promoted elsewhere.

## Core Terms
Cognitive Specification Layer
A technology-agnostic layer that specifies what cognition should exist rather than how cognition executes. In this source, UMG is explicitly locked as a CSL.

Thought Unit
The raw semantic idea before MOLT typing.

MOLT Block
A typed Thought Unit with role and metadata.

PrimaryShell
The single output anchor per Snap, defined as a role and explicitly not the same thing as MOLT Primary.

## Structural Terms
Module
A higher-level human-facing container grouping multiple MOLT Blocks for reuse.

Composite Block / Macro
Authoring-layer convenience that expands into multiple MOLT Blocks before compilation.

Compiler Stack
Mechanical same-type resolution / ordering concept inside the compiler.

Organizational Stack
Human-facing grouped cognitive dimension such as personality, legal, or strategy.

## Definitional Candidates
- Universal Modular Generation is a tech-agnostic Cognitive Specification Layer.
- MOLT is the formal grammar of UMG.
- MOLT Primary is not Role PrimaryShell.
- Modules group blocks for humans; MOLT Blocks are what the compiler understands.
- Stacks organize thinking. PrimaryShell defines output.
