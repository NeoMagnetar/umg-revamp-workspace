# BATCH-049 — Definitions Extract

## Identity / Sleeve Language
- **Sleeve**: the active identity/configuration container; who the system is right now.
- **Resleeving**: switching or redefining the active sleeve/configuration.
- **Reference architecture**: library or model of structure, not a literal costume or backend claim.

## Architecture Layers
- **NeoStack**: domain/mode-level container of grouped functional behavior.
- **NeoBlock**: grouped active functional unit under a NeoStack.
- **MOLT atom / Block**: primitive explanatory/compiler-level semantic unit.

## Governance / Authority
- **Sleeve governance**: local constraints/defaults attached to the active sleeve structure.
- **External governance / substrate authority**: higher-order or immutable constraints not truthfully reducible to sleeve-local design.
- **Bounded agency**: agency constrained by governance locks, validation, scope limits, audit trace, and loop bounds.

## Runtime / Compiler Language
- **Deterministic dominance ordering**: repeatable conflict/order resolution using stable sorting criteria.
- **Validation schema**: rules that can reject or accept a structure at compile/activation time.
- **RuntimeSpec**: ordered agency/runtime loop specification.
- **Safety Layer Pack**: governance/safety adjunct architecture for bounded runtime behavior.
- **Behavioral Abstraction Model**: publication-safe label for conceptual behavior mapping that avoids implying literal backend access.

## Output / Rendering Language
- **MODE**: how cognition is configured/selected.
- **BP_MODE / format blueprint**: how output is emitted/rendered.
- **Orthogonal**: cognition selection and emission format are independent layers.

## Continuity / Memory Language
- **Reconstruction is not memory**: warm-start or documentation-based continuity is not the same as literal persistent recollection.
- **Session continuity tension**: unresolved issue of how to represent cross-session persistence honestly.

## Open Definition Tensions
- priority semantics
- inter-sleeve scope rules
- external governance placement
- continuity semantics
