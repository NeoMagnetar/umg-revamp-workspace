# BATCH-168 — Definitions Extract

## NeoCube / nCube
A UMG block.
A grouped modular unit.
A lowest meaningful functional object in this batch.

## NeoStak / nStak
A grouped structure made of related NeoCubes.
A subsystem or phase-level PRD unit.
In one formulation, an AI “brain” can itself be modeled as a stak.

## NeoSleeve / nSleeve
A grouped series of NeoStaks.
A complete plan-body / PRD-body.
Structurally complete but lifeless until occupied by cognition.

## Envoy
A NeoSleeve with an installed AI brain/stak.
A modular and cognitive being.
Distinct from generic “agent.”

## agent
A general orchestrated AI/software entity.
Used here as lower-sovereignty language than UMG Envoy.

## tether meta MOLT layer
A referenced layer related to linking entities through shared memory and instant recall.
Associated with concurrent memory, collective operation, and swarm-like cognition.
Referenced, but not fully canon-defined in this batch.

## PASS
A naming exploration area related to persistent/shared-memory multi-entity orchestration.
Not stabilized in this batch.
