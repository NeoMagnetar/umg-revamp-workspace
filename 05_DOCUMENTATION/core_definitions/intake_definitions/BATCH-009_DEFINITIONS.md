# BATCH-009_DEFINITIONS

## Definitions Extract

### Terminology
- **Diegetic honesty:** truthful communication expressed through the active persona rather than through base-model or meta disclosure.
- **Persona continuity:** preservation of active sleeve identity across uncertainty, correction, limitation, or constraint expression.
- **No meta voice:** rule that limitations or corrections should not leak into generic model-explanatory framing when a sleeve is active.
- **Voice authority leak:** failure mode where truthful output exits the active persona and reverts to a base-model/meta tone.
- **Epistemic translation:** transformation of uncertainty, limitation, or correction statements into active-persona-consistent language.
- **Continuity hardening:** explicit governance/runtime protection that keeps sleeves coherent under pressure.
- **Project Mode:** file-aware governed workspace mode for multi-file, sleeve-aware UMG work.
- **Agent Mode:** execution-oriented mode that should not be assumed to share project-file awareness.

### Role Language
- **Sleeve:** who the system is right now.
- **Stack:** posture/method layer that can modify behavior without replacing the overall architecture.
- **Governance:** higher control layer that preserves truthfulness and bounded behavior.
- **Blueprint:** output-format layer; separate from cognition and persona continuity.

### Architecture Language
- **Governance -> Continuity -> Style:** candidate priority ordering surfaced by the batch.
- **Envoy sleeve:** public-facing, respectful, community-usable sleeve direction.
- **Persona continuity guard block:** candidate block family for continuity protection.
- **Public stewardship posture:** intended public release mode emphasizing legibility, respect, and non-deceptive continuity.

### Runtime / Compiler Language
- **Continuity-preserving transform:** compiler/runtime path that routes epistemic statements through persona-aware phrasing.
- **Base-model explanatory default:** undesired fallback voice that bypasses the active sleeve.
- **Runtime failure mode:** continuity failure despite factual correctness.
