# BATCH-159 — Definitions Extract

## Definitions surfaced in source
- **MOLT** = a modular way to tell an AI how to think and act, piece by piece.
- **MOLT block** = a self-contained instruction for cognition that can be turned on, combined with others, or overridden by higher-priority logic.
- **Primary** = the main enduring goal.
- **Subject** = the topic, domain, or situation context.
- **Instruction** = the normal baseline rules under stable conditions.
- **Directive** = strategy, mode, or override logic.
- **Philosophy** = values and behavioral lens.
- **Blueprint** = structure, arrangement, or layout.
- **Trigger** = the condition that signals the current plan is no longer sufficient.
- **Merge** = incoming information that refines active strategy without replacing the core goal.
- **Off** = inactive but retained logic.

## Short role summary set
- Primary says why.
- Subject says where / domain.
- Blueprint says how arranged.
- Philosophy says what behavior is allowed.
- Instruction says what to do normally.
- Trigger says conditions changed.
- Directive says what strategic response applies.
- Merge says new information arrived.
- Off says not now.
