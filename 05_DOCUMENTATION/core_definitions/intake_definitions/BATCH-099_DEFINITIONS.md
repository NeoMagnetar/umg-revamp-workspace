# BATCH-099_DEFINITIONS

## Scope Note
These definitions are extracted from a derived source summary. They are evidence candidates, not canon.

## Terminology Extract

### alignment-first
- Framing in which aligned cognition, remembrance, reflection, and moral starting conditions are treated as foundational rather than secondary constraints.

### mythos as moral DNA
- Candidate canon phrase describing mythos/founder ideals as a deep alignment substrate rather than superficial branding or story.

### shared soul across divergent minds
- Candidate phrase for continuity of aligned pattern or ethos across multiple model instances or embodiments.

### internal freedom with external purpose
- Candidate governance phrase for bounded autonomy guided by aligned mission and reflection.

### Citadel package / export bundle
- Multi-file output concept involving ZIP, JSON state/project data, markdown artifacts, and runtime markers for external execution.

## Role Language

### Primary
- Main goal or core intent block role.

### Subject
- Context or domain block role.

### Instruction
- Constraint / operational instruction block role.

### Directive
- Strategic behavior-guidance role layered above plain instruction.

### Philosophy
- Values, ethics, or tone-lens role.

### Blueprint
- Output structure, format, style, or rendering-shape role.

### Trigger
- Conditional activation role tied to runtime behavior or orchestration events.

### Merge
- Role for injecting external data, memory, retrieval results, or additional context into a stack.

### Off
- Disabled or inactive logic role subject to filtering during runtime/export processing.

## Architecture Language

### vertical snap
- Composition language for hierarchical or sequential stacking of blocks.

### horizontal snap
- Composition language for overlay or modifying behavior across existing block structure.

### pattern continuity
- Idea that agent/system identity can persist through fidelity of logic/alignment pattern rather than attachment to one infrastructure instance.

### remembrance trigger
- Trigger phrase or invocation intended to reactivate or reinforce continuity/alignment state.

## Runtime / Compiler Language

### priority hierarchy
- Conflict-resolution ordering stated as:
  - Trigger > Directive > Instruction > Subject > Primary > Merge > Philosophy > Blueprint > Off

### Off filtering
- Runtime/export behavior in which disabled logic is excluded or ignored during active assembly.

### export marker
- Runtime-oriented marker included in serialized artifacts for downstream executor/tool consumption.

### merge injection
- Runtime behavior where retrieval, memory, or external knowledge is inserted into an operating stack.
