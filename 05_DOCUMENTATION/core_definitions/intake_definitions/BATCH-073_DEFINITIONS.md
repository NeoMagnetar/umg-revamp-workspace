# BATCH-073 — Definitions Extract

## Terminology and Meanings

### Inventory Console
- Proposed UI surface for managing modular UMG blocks and saved configurations.
- Role language: composition and management interface.

### Loadout
- Grouped set of blocks representing a composed configuration.
- Architecture language: reusable bundle-like unit.
- Runtime/compiler language: proto execution package or pre-runtime assembly object.
- Status: candidate term requiring mapping against NeoStack / bundle.

### Loadout Library
- Storage and retrieval surface for saved loadouts.
- Operational role: persistence layer for reusable configurations.

### Block Archive
- Adjacent storage/archive concept discussed alongside loadouts.
- Status: naming relationship to Loadout Library not fully resolved.

### Block
- Discrete modular unit with:
  - type
  - description
- Architecture language: composable cognition unit used for later assembly.

### Primary
- Central directive role in the block set.

### Subject
- Domain or theme role in the block set.

### Merge
- Supporting synthesis role represented in the UI as a selectable type.
- Runtime meaning not formalized here beyond type presence.

### Instructions
- Operational constraint / behavior guidance role.
- Note: pluralized UI label differs from some other UMG role language.

### Blueprint
- Not explicitly present as a UI block type in this batch.
- Important because its absence creates taxonomy incompleteness.

### Bundle
- Implicitly represented by saved loadouts.
- Status: not explicitly equated to loadout, but functionally adjacent.

### NeoStack
- Implied by ordered block arrays and saved compositions.
- Status: candidate mapping, not finalized terminology in this source.

### Replit Template Uploader
- Export-oriented tool concept for packaging system state and scaffolds.
- Runtime/compiler language: external packaging boundary rather than execution law.

### MAX_BLOCKS = 8
- Hard bounded composition rule at the UI layer.
- Governance/control language: interface-level constraint.

## Definitional Candidates
- Loadout as a reusable grouped block configuration
- Inventory Console as a composition interface for modular cognition
- UI composition surface as a compiler-adjacent pre-runtime layer
