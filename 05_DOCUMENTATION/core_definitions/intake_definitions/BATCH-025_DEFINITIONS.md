# BATCH-025 Definitions Extract

## Terminology
### Bundle
An ordered grouping of MOLT blocks of the same type. It preserves identity and local precedence rather than synthesizing a new doctrine.

### Merge
A rare, high-significance synthesis operation. Unlike Bundle, Merge creates a more unified identity or doctrine.

### NeoBlock
A compiled operational unit that may contain standalone MOLT blocks and/or Bundles.

### Sleeve
The active identity context that encloses defaults, constraints, overlays, and runtime architecture. Not just a title header.

## Meanings
- **Activation does not erase precedence:** Triggered or routed behavior still remains subordinate to higher-order doctrine.
- **Lower-priority blocks operate within the bounds of higher-priority blocks:** Operational logic is bounded by upstream rules.
- **Exceptions must be declared at or above the level of the rule they qualify:** No silent exception injection from lower layers.

## Role Language
- **Trigger / gate:** Opens a path or situational branch.
- **Directive / route:** Selects posture or path family under existing doctrine.
- **Bundle order:** Local precedence for same-type grouped blocks.

## Architecture Language
- MOLT block → NeoBlock → NeoStack → Sleeve is treated as canon-strength hierarchy.
- Bundle lives inside larger architectural contexts while respecting the higher-order hierarchy.
- Packaging should not freeze unresolved doctrinal ambiguity too early.

## Runtime / Compiler Language
- Gates control activation.
- Directives control routing/posture.
- Bundle order controls local same-type precedence within the active set.
- Compiler/runtime doctrine should preserve a three-layer pattern:
  - top doctrine
  - middle route activation
  - lower operational execution

## Definitional Candidates
- “A Bundle is an ordered grouping of MOLT blocks of the same type.”
- “Activation does not erase precedence.”
- “Lower-priority blocks operate within the bounds of higher-priority blocks.”
- “Exceptions must be declared at or above the level of the rule they qualify.”
