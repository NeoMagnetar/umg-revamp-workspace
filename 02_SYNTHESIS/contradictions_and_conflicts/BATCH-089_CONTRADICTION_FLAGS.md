# BATCH-089 — Contradiction / Ambiguity Flags

## Contradiction Watch
HIGH

## Main Flags
### 1. MCA Draft Status
MCA is explicitly non-canonical and draft, so it should not be promoted automatically.

### 2. AGI Comparison Ambiguity
The batch positions MCA as an alternative to AGI framing, but does not settle whether MCA replaces, approximates, or subsets AGI concepts.

### 3. Negentropy Formality
Negentropy is mapped conceptually, but not integrated into a formal UMG specification.

### 4. Balanced Soul Doctrine
Referenced as guidance, but not formally defined in the source summary.

### 5. Hidden Instruction Governance
Hidden instruction blocks are powerful conceptually, but the rules governing their visibility, validation, and safety are not formalized.

### 6. Identity vs Configuration Boundary
The source implies identity is configurable via block injection, but does not fully separate stable identity from temporary configuration.

## Intake Conclusion
Use this batch for:
- taxonomy discussion
- glossary candidate review
- identity/stack distinction work
- hidden instruction behavior notes

Do not use it alone to settle:
- MCA canon status
- AGI replacement claims
- final hidden-instruction runtime law
- final negentropy doctrine
