# UMG Revamp Workspace

This repository mirrors the local UMG Revamp Workspace as a readable public corpus.

## Source-of-truth rule
The local workspace remains the canonical working surface. This GitHub repository is a public mirror intended for navigation, readability, front-end GPT use, and OpenClaw-assisted workflows.

## Mirroring approach
- Preserve the existing top-level workspace lanes.
- Keep readable files in readable paths where practical.
- Do not treat `UNZIPPED FILES` as a primary public knowledge lane.
- Do not assume semantic consolidation has already happened.
