# Intake Review Standard

## Review Format
Each intake batch should be processed into:
1. Source Snapshot
2. Chat-Level Summary
3. UMG-Relevant Extraction
4. Independent Review
5. Roadmap Mapping
6. Action Outcome

## Batch Output Shape
- Batch Overview
- Per-File Summary Cards
- Cross-File Synthesis
- Roadmap Impact
- Recommended Next Decision
- Checklist Update Block
