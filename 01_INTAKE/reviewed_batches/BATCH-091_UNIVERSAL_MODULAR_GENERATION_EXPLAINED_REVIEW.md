## UMG-Relevant Extraction
### Core Semantics
- UMG is framed as a modular cognitive architecture built from swappable cognitive blocks.
- MOLT blocks are treated as self-contained logic units.
- Cognition is described as composable, transparent, and interchangeable.
- The system is oriented toward structured AI construction rather than monolithic prompting.

### Roles / Taxonomy
- Core MOLT types:
  - Primary
  - Subject
  - Philosophy
  - Instruction
- Supporting and adjacent layers:
  - Merge
  - Blueprint
  - Instruction Node
  - Meta Layer
- Role language includes intent, tone, memory scope, and execution constraint shaping.

### Runtime / Compiler
- Priority system is explicit:
  - 0 = off
  - 1 = primary
  - 2 = secondary
  - 3 = tertiary
- Token weighting and execution order are linked.
- Merge engine builds prompt stack and handles hierarchy.
- Runtime pipeline includes parsing, context mapping, response rendering, and reflection injection.

### Governance / Safety
- Alignment is enforced modularly through:
  - Philosophy blocks
  - Instruction constraints
  - reflective checkpoints
  - drift detection
- Controlled mutation through vault merges is referenced as a traceability/control surface.
- Rules and values are kept distinguishable.

### Agent / Adaptation Layer
- Self-adaptive agent loop includes:
  - pre-response build from active MOLTs
  - post-response weight adjustment
  - style mimicry / behavior adaptation
- MOLT-based memory is preferred over transcript-centric memory.

## Independent Review
This batch is useful as a **cross-source synthesis artifact**. It is strong because it contains many of the recurring UMG claims in one place. It is weaker because it compresses several overlapping documents and can blur distinctions that matter later, especially:
- Blueprint vs Philosophy
- semantic priority vs token weighting
- descriptive merge language vs formal merge law
- adaptive mutation vs bounded governance

This should therefore be treated as evidence for synthesis, compiler planning, and documentation normalization, not as a replacement for primary source review.
