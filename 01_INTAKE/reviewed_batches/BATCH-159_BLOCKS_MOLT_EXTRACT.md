# BATCH-159 — Blocks / MOLT Extract

## MOLT role extraction
### Primary
- Enduring top-level aim
- Other blocks serve it

### Subject
- Situation/domain framing
- Establishes contextual assumptions

### Instruction
- Baseline orders under normal conditions
- Stable while assumptions hold

### Directive
- Strategic mode / override logic
- Appears in two forms:
  - reactive override
  - pre-loaded doctrine

### Philosophy
- Behavioral restraint / value lens

### Blueprint
- Structural arrangement
- Separate from ethics and strategy

### Trigger
- State-change detector
- Does not itself decide the response

### Merge
- New information refining active strategy
- Does not replace Primary

### Off
- Stored inactive logic
- Available for later activation

## Key semantic distinction
The source implies a meaningful difference between:
1. Directive as crisis-time override
2. Directive as standing doctrine already loaded before crisis

This should be preserved as evidence rather than silently harmonized.
