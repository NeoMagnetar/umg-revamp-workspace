# BATCH-174 — Blocks / MOLT Extract

## Operative Builder Categories
- Primary
- Subject
- Instruction
- Philosophy
- Directive

## Source-Faithful Structural Rule
Top level = MOLT type  
Inside each type = selectable blocks organized under that type

## MVP Quantity Constraint
At least 10 starter blocks in each listed MOLT category.

## Category Usage in This Batch
### Primary
Core website layout / structure elements.

### Subject
Page/content/domain units.

### Instruction
Behavior, rendering, logic, or interaction helpers.

### Philosophy
Aesthetic or style overlays.

### Directive
Strategic, optimization, policy, or operational behaviors.

## Important Limitation
These categories are operative for this builder pass, but this batch does not establish that they are the full or final UMG-wide category set.
