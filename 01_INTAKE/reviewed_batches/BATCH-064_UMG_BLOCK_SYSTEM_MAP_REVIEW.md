## Independent Review
This is a high-value evidence batch because it does not merely restate block types — it maps how multiple UMG documentation layers are currently being used together in practice. The strongest stable signals are the Primary / Merge / Blueprint role separation, the MOLT taxonomy, the active-stack example, and the runtime execution sequences. The strongest unstable signals are Cantocore scope, overlay rules, and the incomplete formalization of many terms that the user wants ready for long-term memory or persistent instructions. This batch should be used as evidence for mapping and reconciliation, not as a final canonical schema.
